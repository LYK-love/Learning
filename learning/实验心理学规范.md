## 实验室建设

### 屏蔽处理

对*认知神经科学*技术仪器设备的使用必须隔离外界的磁场，当外界磁场无法避免时，就必须建立屏蔽室，以屏蔽外界磁场来使用磁共振仪、ERP等设备

### 消隔音处理

消隔音的标准：

1. A类设计：背景噪声在25db(A)以下（这是规范中医院“听力测试室”的一级标准）
2. B类设计：背景噪声在25db(A)以下（这是规范中医院“听力测试室”的二级标准）
3. 类设计：背景噪声在35db(A)以下（这是规范中医院“听力测试室”的三级标准）

*白噪音是40db*

### （四）正念
* 近年来关于正念的研究数量剧增
派别：
> 佛教正念：
>
> 理论：以小乘佛教为代表
>
> 
>
> 西方正念---Kabat-Zin
>
> * 正念是以一种特殊的方式集中注意力，有意的，在当下的，不带评判的
> * 核心观点：对注意力的自我调节
> * 意识可以指向内部线索(例如身体的感觉)，也可以指向外部的线索（如外部的环境或新事物）
> * 实践：正念减压疗法
> * MBSR:东方禅修冥想和西方心理教育相结合
> * 其它变式：正念认知疗法；正念复发预防；正念饮食训练
> * 正念训练可以显著缓解焦虑
>
> 
>
> 西方正念--Ellen-Langer
>
> * 正念与非正念
>
> * 社会认知取向：对新鲜事物的开放性，灵活的思维和认知重建
>
> * 接受不确定性：任何解决问题的过程都可以有多个视角
>
> * 可通过不同方式达到正念状态：如注意差别；多角度思维；创造新奇性等
>
> * Langer正念易于实现，尤其适用于不适合冥想的群体
>
> * 对提升学习和认知表现有效。
>
>   
>
> 两种正念的异同：
>
> 正念冥想通常将正念视为一种结果(即通过冥想，达到正念的状态)；而Langer流派的正念则聚焦于过程
